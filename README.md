MybatisSpring2.5.X
==================

Spring2.5.X integration for MyBatis 3

由于现有系统基于Spring2.5.6开发,升级到Spring3有一定风险,并且有很大的时间成本.
Mybatis3又没有兼容Spring2.5系列的集成实现.
鉴于以上情况,参考了Mybatis Spring的实现进行改造,完成了这个框架.

为了方便其他有此需求的朋友们,将这个框架放到了GitHub上.
